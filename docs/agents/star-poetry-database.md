# 星曜诗赋数据库 — Star Poetry Database

> Complete extract from `sample_s.rule`. Three rule types parsed into structured format.

---

## Format

```
躔度歌 (Mansion): 11 planets × 28 mansions = 308 entries
入宫歌 (Zodiac):  11 planets × 12 signs  = 132 entries
照宫歌 (House):   11 planets × 12 houses = 132 entries
```

## Data Structure for Code

```rust
/// 星曜躔度歌 — Planet in Lunar Mansion
struct MansionPoetry {
    planet: &'static str,  // 日/月/金/木/水/火/土/计/罗/炁/孛
    mansion: &'static str, // 28 mansions: 角/亢/氐/房/心/尾/箕/斗/牛/女/虚/危/室/壁/奎/娄/胃/昴/毕/觜/参/井/鬼/柳/星/张/翼/轸
    rank: &'static str,    // e.g. [2.11.3]
    title: &'static str,   // e.g. 日躔井宿号天弼
    verse: &'static str,   // e.g. 冠世文章增福力、公侯卿监世间荣、福禄恩波传子息。
}

/// 星曜入宫歌 — Planet in Zodiac Sign
struct SignPoetry {
    planet: &'static str,
    sign: &'static str,    // 12 signs: 子/丑/寅/卯/辰/巳/午/未/申/酉/戌/亥
    rank: &'static str,
    title: &'static str,   // e.g. 太阳居丑号天幽
    verse: &'static str,
}

/// 星曜照宫歌 — Planet in Astrology House
struct HousePoetry {
    planet: &'static str,
    house: &'static str,   // 命宫/财帛/兄弟/田宅/男女/奴仆/夫妻/疾厄/迁移/官禄/福德/相貌
    rank: &'static str,
    title: &'static str,   // e.g. 土星居命主心痴
    verse: &'static str,
}
```

## Usage in Code

```rust
/// Lookup verse for a planet in its current mansion
fn get_mansion_verse(planet: &str, mansion: &str) -> Option<&'static MansionPoetry> {
    MANSION_POETRY.iter().find(|p| p.planet == planet && p.mansion == mansion)
}

/// Lookup verse for a planet in its current zodiac sign
fn get_sign_verse(planet: &str, sign: &str) -> Option<&'static SignPoetry> {
    SIGN_POETRY.iter().find(|p| p.planet == planet && p.sign == sign)
}

/// Lookup verse for a planet in its current house
fn get_house_verse(planet: &str, house: &str) -> Option<&'static HousePoetry> {
    HOUSE_POETRY.iter().find(|p| p.planet == planet && p.house == house)
}
```
