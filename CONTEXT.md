# Moira

中国传统占星学应用。支持七政四馀、二十八宿占星、紫微斗数三大体系的星盘排盘与解读。

## 架构

Tauri v2 桌面应用。Rust 库 (`moira-core`) 负责天文计算和排盘逻辑，React + Vite 前端负责 UI 渲染。

## Language

**星盘** (Xīngpán):
用户的出生天宫图，包含星曜、宫位、神煞等全部元素的综合布局。

**七政四馀** (Qīzhèng Sìyú):
基于真实天体位置的传统占星体系。七政指日月+水金火木土七颗行星，四馀指罗睺、计都、月孛、紫炁四颗隐曜。
_避免_: 西洋占星

**二十八宿** (Èrshíbā Xiù):
中国传统的二十八个月站（宿），沿黄道分布，每宿对应特定度数范围。用于七政和择日。
_避免_: 西方星座

**神煞** (Shénshà):
星盘中的吉凶神煞标志，指示特定时空组合对应的吉凶性质。

**星曜**:
星盘中的各类星体。包括七政（日月水金火木土）、四馀（罗计月紫）、紫微斗数虚星等。

**宫位**:
星盘中划分的十二个区域。不同体系对宫位的命名和含义不同。

**JPL DE 星历**:
NASA 喷气推进实验室发布的太阳系天体位置数据。Moira 使用 DE440/DE441 进行高精度行星位置推算。

## Relationships

- **七政四馀** 使用 **JPL DE 星历** 计算 **星曜** 的真实位置 → 确定其在 **二十八宿** 中的分度 → 结合 **神煞** 排盘
- **紫微斗数** 根据出生时间查农历 → 排 **虚星**（紫微、天府等）→ 填入 **十二地支宫** → 结合 **神煞** 解读
- **二十八宿占星** 依赖 **七政四馀** 的星曜位置 → 标注每颗星曜在 **二十八宿** 中的落点

## Flagged ambiguities

- "Moira" 是希腊命运女神名，项目内容为中中国传统占星 — 已决定保留作为开发代号，不做面向用户品牌
- "神煞" 在不同体系中定义不同：七政有七政的神煞表，紫微有紫微的神煞表 — 需分开维护
